
## 1. Фильтрация словаря по порогу значений

Реализуйте функцию `filter_by_value(data, threshold=0)`, которая принимает на вход словарь (ключи — строки, значения — числа) и необязательный аргумент `threshold` (порог отсечения, по умолчанию `0`).  

Функция должна возвращать новый словарь, включающий только те пары ключ-значение исходного словаря, где значение **больше или равно** заданному порогу.

```python
scores = {'Alice': 85, 'Bob': 42, 'Charlie': 100, 'Diana': 37}

print(filter_by_value(scores))
# {'Alice': 85, 'Bob': 42, 'Charlie': 100, 'Diana': 37}

print(filter_by_value(scores, 50))
# {'Alice': 85, 'Charlie': 100}

print(filter_by_value(scores, 90))
# {'Charlie': 100}

print(filter_by_value(scores, 42))
# {'Alice': 85, 'Bob': 42, 'Charlie': 100}
```

## 2. Объединение нескольких словарей

Напишите функцию `merge_many_dicts`, которая принимает **произвольное количество словарей** (`*dicts`) и возвращает один словарь, объединяя их.  

При совпадении ключей значение берётся из **последнего** словаря (перезаписывается).

```python
d1 = {'a': 1, 'b': 2}
d2 = {'b': 3, 'c': 4}
d3 = {'c': 5, 'd': 6}

print(merge_many_dicts(d1, d2, d3))
# {'a': 1, 'b': 3, 'c': 5, 'd': 6}

print(merge_many_dicts(d1, d3))
# {'a': 1, 'b': 2, 'c': 5, 'd': 6}
```

## 3. Создание словаря с возможностью реверса

Напишите функцию `make_dict`, которая принимает **произвольное количество именованных аргументов** (`**kwargs`) и возвращает словарь.  

Добавьте дополнительную опцию: если передан булев параметр `reverse` (по умолчанию `False`), то ключи и значения меняются местами.  

- Если `reverse=True`, но значения не могут быть ключами (например, список), функция должна вернуть `None` (или выбросить исключение — на ваше усмотрение).

```python
print(make_dict(name='Alice', age=30, city='Moscow'))
# {'name': 'Alice', 'age': 30, 'city': 'Moscow'}

print(make_dict(reverse=True, name='Alice', age=30, city='Moscow'))
# {'Alice': 'name', 30: 'age', 'Moscow': 'city'}

# Попытка реверса со значениями, которые не могут быть ключами (например, список)
# print(make_dict(reverse=True, data=[1,2,3]))  # Вернёт None
```
