# Задача 1: Слияние двух отсортированных списков

## Условие

Даны два списка целых чисел `list1` и `list2`, каждый из которых отсортирован по возрастанию. Необходимо объединить их в один отсортированный список.

Реализуйте функцию `merge_sorted(list1, list2)`, которая возвращает новый список, содержащий все элементы обоих списков в порядке неубывания (т.е. каждый следующий элемент больше либо равен предыдущему).

## Требования

- Запрещено использовать встроенную сортировку (например, `sorted(list1 + list2)`) — нужно реализовать алгоритм самостоятельно.
- Алгоритм должен работать за линейное время **O(n + m)**, где `n` и `m` — длины списков.
- Нельзя модифицировать исходные списки — создаётся новый список.
- Используйте **метод двух указателей** (two pointers) для эффективного слияния.

## Примеры работы

```python
# Пример 1
a = [1, 3, 5, 7]
b = [2, 4, 6, 8, 10]
print(merge_sorted(a, b))  
# Ожидается: [1, 2, 3, 4, 5, 6, 7, 8, 10]

# Пример 2 (списки разной длины)
c = [1, 2, 3]
d = [4, 5, 6, 7]
print(merge_sorted(c, d))  
# Ожидается: [1, 2, 3, 4, 5, 6, 7]

# Пример 3 (один из списков пустой)
e = []
f = [1, 2, 3]
print(merge_sorted(e, f))  
# Ожидается: [1, 2, 3]

# Пример 4 (равные элементы)
g = [1, 3, 5]
h = [1, 2, 3, 4]
print(merge_sorted(g, h))  
# Ожидается: [1, 1, 2, 3, 3, 4, 5]
```

## Подсказка

Алгоритм слияния двух отсортированных списков работает следующим образом:

1. Создайте два указателя (индекса): `i` для первого списка и `j` для второго, оба начинаются с 0.
2. Создайте пустой список `result` для хранения результата.
3. Пока оба указателя не вышли за границы соответствующих списков:
   - Сравните элементы `list1[i]` и `list2[j]`.
   - Меньший (или равный) элемент добавьте в `result`.
   - Сдвиньте указатель того списка, откуда взяли элемент.
4. Когда один из списков закончится, добавьте все оставшиеся элементы другого списка в `result`.

# Задача 2: Числа Фибоначчи (итеративный способ)

## Условие

Числа Фибоначчи — это последовательность, в которой каждое следующее число равно сумме двух предыдущих.  
Классическая последовательность начинается с чисел 0 и 1:

- F(0) = 0
- F(1) = 1
- F(n) = F(n-1) + F(n-2) для n ≥ 2

В этой задаче вам нужно реализовать функцию `fibonacci(n, first=0, second=1)`, которая вычисляет **n-й** элемент последовательности Фибоначчи, где первые два числа могут быть заданы пользователем.

Напишите функцию `fibonacci(n, first=0, second=1)`, которая принимает три аргумента:

- `n` — целое неотрицательное число, индекс искомого элемента;
- `first` — значение нулевого элемента последовательности (по умолчанию 0);
- `second` — значение первого элемента последовательности (по умолчанию 1).

Функция должна возвращать n-й элемент последовательности, определённой рекуррентно:

- F(0) = first
- F(1) = second
- F(k) = F(k-1) + F(k-2) для всех k ≥ 2

## Требования

- Запрещено использовать рекурсию — только **итеративный подход** (циклы).
- Функция должна работать для любых неотрицательных `n` и любых целых `first`, `second`.
- Нельзя использовать сторонние библиотеки или встроенные функции, напрямую вычисляющие числа Фибоначчи.

## Примеры работы

```python
# Классическая последовательность
print(fibonacci(10))        # 55
print(fibonacci(5))         # 5
print(fibonacci(0))         # 0
print(fibonacci(1))         # 1

# Обобщённая последовательность: начинается с 2 и 3
# F(0)=2, F(1)=3, F(2)=5, F(3)=8, F(4)=13, F(5)=21, F(6)=34, F(7)=55
print(fibonacci(7, first=2, second=3))   # 55
```

## Подсказка

Для итеративного вычисления удобно хранить два последних числа и обновлять их на каждом шаге:

- Если `n == 0`, вернуть `first`.
- Если `n == 1`, вернуть `second`.
- Для `n >= 2` запустить цикл от 2 до `n`. На каждой итерации новое значение вычисляется как сумма двух предыдущих, после чего сдвигаем переменные.

Пример логики:

```python
prev, curr = first, second
for _ in range(2, n + 1):
    prev, curr = curr, prev + curr
return curr
```
