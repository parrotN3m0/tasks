
## Задача. «Банковский счёт»

**Условие:**  
Создайте класс `BankAccount`, который при создании принимает начальный баланс (`initial_balance`).  
Атрибут `balance` хранит текущую сумму.  
Реализуйте методы:
- `deposit(amount)` — пополняет счёт на указанную сумму (только положительную; если сумма ≤ 0, выводить предупреждение).
- `withdraw(amount)` — снимает указанную сумму, если средств достаточно; иначе выводит сообщение об ошибке.
- `show_balance()` — выводит текущий баланс.

Создайте объект с начальным балансом, например, 1000. Пополните, снимите часть, попробуйте снять больше, чем есть, и каждый раз выводите баланс.

## Задача. «Студент и оценки»

**Условие:**  
Создайте класс `Student` с атрибутами:
- `name` — имя студента;
- `grades` — пустой список (по умолчанию).

Методы:
- `add_grade(grade)` — добавляет оценку в список `grades` (оценка от 1 до 5; при некорректной оценке выводить сообщение).
- `average()` — возвращает средний балл (если оценок нет, вернуть 0).
- `info()` — выводит имя студента и его средний балл.

Создайте двух студентов, добавьте каждому несколько оценок, выведите информацию.


## Задача. «Магазин и корзина»

**Условие:**  
Создайте два класса: `Product` (товар) и `Cart` (корзина).

**Product** имеет:
- `name` (название),
- `price` (цена),
- метод `info()` — возвращает строку `"<name> - <price> руб."`

**Cart** имеет:
- атрибут `items` — пустой список для объектов `Product`.
- метод `add_product(product)` — добавляет товар в корзину.
- метод `total()` — возвращает общую стоимость всех товаров в корзине.
- метод `show()` — выводит список всех товаров (их info) и общую сумму.

Создайте несколько товаров, добавьте их в корзину, покажите содержимое.

### Задача. «Птица» (базовая)

Создайте класс `Bird` с методом `fly()`, который выводит на экран строку `"Птица летит!"`.  
Создайте один объект класса `Bird` и вызовите его метод `fly()`.

**Ожидаемый вывод:**  
```
Птица летит!
```

---

### Задача. «Именованная птица» (использование `__init__` и атрибутов)

Добавьте в класс `Bird` метод `__init__`, который принимает один аргумент `name` и сохраняет его в атрибуте `self.name`.  
Метод `fly()` теперь должен выводить `"<name> летит!"`, где `<name>` — это имя конкретной птицы.  
Создайте двух птиц с разными именами и заставьте каждую полететь.

**Пример использования:**  
```python
sparrow = Bird("воробей")
sparrow.fly()
eagle = Bird("орёл")
eagle.fly()
```

**Вывод:**  
```
воробей летит!
орёл летит!
```

### Задача. «Птица с местом назначения» (метод с аргументами)

Добавьте методу `fly()` возможность принимать аргумент `destination` (куда летит птица).  
Теперь метод должен выводить `"<name> летит в <destination>"`.  
Если место назначения не указано (аргумент по умолчанию `None`), пусть выводится `"<name> летит в неизвестном направлении"`.  
Создайте объект и проверьте оба варианта вызова.

**Пример:**  
```python
parrot = Bird("попугай")
parrot.fly("джунгли")
parrot.fly()
```

**Вывод:**  
```
попугай летит в джунгли
попугай летит в неизвестном направлении
```
