## Задача 1. Прямоугольник

Создайте класс `Rectangle`, который представляет прямоугольник.

### Требования к классу

- Метод `__init__(self, width, height)` — принимает ширину и высоту прямоугольника (числа, могут быть целыми или с плавающей точкой). 
- Метод `area(self)` — возвращает площадь прямоугольника.
- Метод `perimeter(self)` — возвращает периметр прямоугольника.
- Метод `is_square(self)` — возвращает `True`, если прямоугольник является квадратом (ширина равна высоте), иначе `False`.

### Пример использования

```python
rect = Rectangle(4, 5)
print(rect.area())        # 20
print(rect.perimeter())   # 18
print(rect.is_square())   # False

square = Rectangle(3, 3)
print(square.is_square()) # True
```

### Примеры тестов (для самопроверки)

```python

assert r.perimeter() == 10, "Ошибка: периметр 2x3 должен быть 10"
```

#### Тест 2: проверка на квадрат

```python
print("Тест 2 пройден")
```

## Задача 2. Студент

Создайте класс `Student`, который хранит информацию об успеваемости студента.

### Требования к классу

- Метод `__init__(self, name, group)` — принимает имя студента и номер группы.
- Метод `add_grade(self, grade)` — добавляет оценку в список `grades`, **только если** оценка является целым числом от 2 до 5 включительно. В противном случае оценка игнорируется (ничего не добавляется).
- Метод `average_grade(self)` — возвращает среднее арифметическое всех оценок в списке `grades`. Если список пуст, возвращает `0.0`.
- Метод `info(self)` — возвращает строку в формате:  
  `"<Имя>, группа <Номер>, средний балл: <средний_балл>"`.

### Пример использования

```python
s = Student("Анна", "ФИПМ_21")
s.add_grade(5)
s.add_grade(4)
s.add_grade(3)
print(s.average_grade())   # 4.0
print(s.info())            # "Анна, группа ФИПМ_21, средний балл: 4.0"
```

### Примеры тестов (для самопроверки)

#### Тест 1: добавление оценок и средний балл

```python
s = Student("Иван", "ФИПМ_31")
s.add_grade(5)
s.add_grade(4)
s.add_grade(6)   # неверная оценка, не должна добавиться
assert s.grades == [5, 4], "Ошибка: список оценок должен быть [5,4]"
assert s.average_grade() == 4.5, "Ошибка: среднее (5+4)/2 = 4.5"
print("Тест 1 пройден")
```

#### Тест 2: пустой список оценок и формат info

```python
s = Student("Мария", "ФИПМ_11")
assert s.average_grade() == 0.0, "Ошибка: для пустого списка средний балл 0.0"
s.add_grade(5)
s.add_grade(5)
expected_info = "Мария, группа ФИПМ_11, средний балл: 5.0"
assert s.info() == expected_info, "Ошибка: неверный формат строки info"
print("Тест 2 пройден")
```
